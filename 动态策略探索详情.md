# 动态策略探索详情 — AI 量化动量公式对比

> 数据基础：16 只 ETF（宽基+行业+跨境），2021-12-29 ~ 2026-06-26  
> 择时引擎：动态择时（趋势+动量+波动三信号融合）  
> 调仓频率：40 个交易日，Top 2 持仓  

---

## 一、五种动量公式

| 方案 | 公式 | 核心思想 |
|------|------|----------|
| **基准 20日** | `close[-1] / close[-20] − 1` | 单窗口时间序列动量 (MOP 2014) |
| **双动量** | 基准 + 绝对值过滤：`momentum<0 → 踢出候选池` | Gary Antonacci 绝对+相对双动量 |
| **多窗口融合** | `(5日+20日+60日动量) / 3` | Levine & Pedersen 2016 |
| **波动率加权** | 基准 + 仓位 ∝ 1/σ | Barroso & Santa-Clara 2015 |
| **Frog-in-the-Pan** | 信息离散度加权：连续小涨 > 跳跃暴拉 | Da, Gurun, Warachka 2014 |

## 五、关键发现

### 1. 动态择时 → 排名全变了

| 无择时 | 有择时 |
|--------|--------|
| Frog 排第 1 (+12.2%) | Frog 掉到第 3 (+55.0%) |
| 基准排第 3 (−7.0%) | 基准排第 2 (+72.8%) |
| 双动量排第 4 (−7.6%) | **双动量跃居第 1 (+76.8%)** |

> 动态择时引擎已经覆盖了 Frog 的正交价值（过滤跳涨/异常波动），Frog 的增量优势被完全吸收。

### 2. 双动量是真正的黑马

- **唯一 2022 年正收益** (+0.2%)：绝对动量门槛在熊市反弹中避开了假信号
- **牛市不输**：2023/2025 与基准持平
- **累计多赚 4%**（+76.8% vs +72.8%）

### 3. 波动率加权在 A 股水土不服

- A 股高波动板块（科创50）= 高动量 = 最该被选中
- 波动率加权反而压低高波动 ETF 仓位 → 自断一臂
- +40.6% vs 基准 +72.8%，差距 32 个百分点

### 4. 多窗口融合时灵时不灵

- 2026 年突然爆发（+20.5%），但 2022-2025 四年全败
- 2022 −6.3%、2023 +0.1% 在最需要动量的时候反而最差
- **不稳定 = 不可信 → 排除**

## 六、结论与建议

### 🏆 当前最优公式：双动量

```
基准 20日动量 > 0 → 进入候选池
基准 20日动量 < 0 → 踢出（不碰跌的 ETF）

候选池内按动量排序 → Top 2 → 动态择时控制仓位
```

### 📊 效果量化

| 指标 | 基准 | 双动量 | 提升 |
|------|:--:|:--:|:--:|
| 总收益 | +72.8% | +76.8% | **+4.0%** |
| 年化 | 13.6% | 14.2% | **+0.6%** |
| 最大回撤 | −10.4% | −10.4% | 持平 |
| 夏普 | 1.10 | 1.14 | **+0.04** |
| 熊市年正收益 | 0/1 | **1/1** | ✅ |

### ⚠️ 排除的方案

| 方案 | 原因 |
|------|------|
| 多窗口融合 | 5 日噪音太大，4/5 年负贡献 |
| 波动率加权 | A 股高波动=高动量，反向操作 |
| Frog-in-the-Pan | 动态择时已覆盖其价值，无增量 |
